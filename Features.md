# KarghaSetu
### Digital Operating System for Handloom Cooperatives

---

# 1. Overview

KarghaSetu is a comprehensive digital platform designed to digitize and modernize the operations of handloom cooperatives and producer groups.

The platform helps cooperatives efficiently manage members, production, procurement, inventory, finance, communication, analytics, and buyer interactions while strengthening collective bargaining.

It consists of:

- Flutter Mobile Application (Weavers)
- React Web Dashboard (Admin, Buyer, Government)

---

# 2. Objectives

The platform aims to:

- Digitize cooperative operations
- Improve production visibility
- Enable transparent record keeping
- Strengthen collective bargaining
- Reduce manual paperwork
- Improve procurement efficiency
- Connect buyers directly with cooperatives
- Improve financial transparency
- Enable data-driven decision making
- Improve livelihood of weaving communities

---

# 3. User Roles

There are four primary stakeholders.

## 1. Weaver

Uses the mobile application to receive work, update production, communicate with the cooperative, and track earnings.

---

## 2. Cooperative Administrator

Manages the complete cooperative including members, production, inventory, procurement, finance, analytics, and buyer relationships.

---

## 3. Buyer

Places bulk orders, communicates with cooperatives, tracks production, and manages purchases.

---

## 4. Government

Monitors registered cooperatives, publishes schemes, verifies records, and analyzes production data.

# 20. Platform Vision

KarghaSetu is designed as a complete digital operating system for handloom cooperatives.

It enables cooperatives to digitize operations, improve collaboration, optimize procurement, manage production, strengthen collective bargaining, provide transparent governance, connect directly with buyers, and improve the economic well-being of weaver communities through a single integrated platform.
