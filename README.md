# KarghaKendra

> **AI-Powered Cooperative Management & Collective Procurement Platform for Weaver Communities**

KarghaKendra is a digital platform designed to empower weaving cooperatives by digitizing their daily operations, enabling collective bargaining, and connecting them with brands, suppliers, and buyers. By combining AI-powered insights with transparent record-keeping, the platform helps reduce operational costs, improve collaboration, and increase market opportunities for artisan communities.

---

## 🌟 Problem Statement

Many weaving cooperatives still rely on manual processes for member management, procurement, inventory, sales, and communication. This results in:

- Inefficient operations
- Poor transparency
- Weak collective bargaining
- Higher procurement costs
- Limited market access
- Lack of data-driven decision making

KarghaKendra addresses these challenges through a centralized digital ecosystem built specifically for the handloom sector.

## ⚡ Getting Started

Clone the repository

```bash
git clone <repository-url>
```

Navigate into the project

```bash
cd karghakendra
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Open your browser at

```
http://localhost:5173
```
